# LEE 電力モニター

日本の電力市場データを一画面で確認できる Windows デスクトップアプリです。  
Google OAuth によるホワイトリスト認証を採用しており、承認済みユーザーのみが利用できます。

---

## 主な機能

### ⚡ 電力データ

| ウィジェット | データソース | 概要 |
|---|---|---|
| **ダッシュボード** | 各ウィジェット集約 | 主要指標をワンビューで確認 |
| **JEPX スポット市場** | JEPX 公式 | システム〜九州 10エリアの当日・日次・月次・年次・曜日別平均価格 |
| **電力予備率** | OCCTO 公式 | 全エリア予備率の一覧表示・ヒートマップ・警告アラート |
| **インバランス単価** | OCCTO 公式 CSV | 変動単価・グラフ・テーブル・CSV ダウンロード |
| **JKM LNG 価格** | Yahoo Finance (`JKM=F`) | 2014年〜現在の LNG スポット価格推移 |
| **全国天気** | Open-Meteo API | 全国 10 地域の週間天気予報 |
| **発電稼働状況 (HJKS)** | JEPX HJKS | 全国 10 エリアの電源種別稼働・停止容量 (14 日間) |

### 🔵 Google 連携

| ウィジェット | 概要 |
|---|---|
| **Google カレンダー** | 複数カレンダーの表示・イベント作成・編集・削除・ドラッグ移動 |
| **Gmail** | ラベル別メール一覧・本文プレビュー・既読管理・新着ポーリング通知 |

### 🛠 ツール

| ウィジェット | 概要 |
|---|---|
| **AI チャット** | Gemini / Groq による AI アシスタント |
| **通知センター** | アプリ内アラートを一覧表示 (SQLite 永続化) |
| **テキストメモ** | 自由記述メモ (ローカル保存) |
| **バグ報告** | メール送信による不具合報告 |
| **ログビューア** | アプリログのリアルタイム表示 |

---

## 動作要件

- **OS**: Windows 10 / 11 (64bit)
- **Google アカウント**: Google 連携機能の利用に必要
- インストール不要・ランタイム不要 (exe 一体型)

---

## インストール & 起動

1. [最新リリース](https://github.com/jwlee-muji/lee-kojin-app/releases/latest) から `LEE電力モニター.exe` をダウンロード
2. 任意のフォルダへ配置してダブルクリックで起動
3. Google アカウントでログイン (管理者による承認が必要)

### SHA256 整合性確認 (任意)

```powershell
(Get-FileHash 'LEE電力モニター.exe' -Algorithm SHA256).Hash.ToLower()
```

ダウンロードページの `.sha256` ファイルと一致することを確認してください。

---

## 認証・アクセス管理

- ログインは **Google OAuth 2.0** で行います
- 利用可能なアカウントは管理者が **Google スプレッドシート** でホワイトリスト管理
- 未登録アカウントはアクセス申請メールを送信できます

---

## 自動アップデート

起動時に GitHub Releases API で最新バージョンを確認します。  
新バージョンがある場合はダイアログで通知し、承認するとダウンロード・上書き・再起動が自動で行われます。

---

## 設定

設定画面から以下を変更できます。

- 各ウィジェットの自動更新間隔・警告閾値
- テーマ (ダーク / ライト)
- 表示言語 (日本語 / 英語 / 韓国語 / 中国語)
- Google 連携の認証・解除
- ユーザー管理 (管理者のみ)
