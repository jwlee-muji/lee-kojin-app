# LEE 電力モニター

日本の電力市場データをリアルタイムで確認できるデスクトップアプリです。

## 機能

- **電力予備率** (OCCTO) — エリア別予備率を日付指定・30分自動更新
- **インバランス単価** — CSV取得・グラフ表示・30分自動更新・色分け表示

## 動作要件

- Windows 10 / 11 (64bit)
- **Google Chrome** がインストールされていること（予備率スクレイピングに使用）

## インストール

1. [Releases](https://github.com/jwlee-muji/lee-kojin-app/releases/latest) から `LEE電力モニター.exe` をダウンロード
2. 任意のフォルダに配置してダブルクリックで起動
3. 初回起動時はインバランス単価タブの「DB更新」ボタンを押してデータを取得

## 自動アップデート

起動時に自動でバージョンを確認します。  
新しいバージョンがある場合はダイアログで通知し、承認するとダウンロード・置換・再起動が自動で行われます。
